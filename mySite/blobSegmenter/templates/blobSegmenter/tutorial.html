BlobSegmenter tutorial

Installing ImageJ and BlobSegmenter

First, install ImageJ. You can download ImageJ here.

Next, download blobSegmenter here and place the .jar file in the "plugins" folder of ImageJ.

Finally, download LibSVM and MorphoLibJ, which are two libraries required by blobSegmenter (libsvm, morpholibj). Place LibSVM and MorphoLibJ in the "plugins" folder of ImageJ.
Note that I've hosted libsvm and morpholibj on my server in case future updates break backwards compatability with blobSegmenter; for completeness, the original .jar files can be found here and here.

That's it, BlobSegmenter should be installed!

Download example datasets

Let's download some sample datasets so we can give BlobSegmenter a spin.

Download the example datasets here and unzip it. You should see a base folder "example_working_directory" with a subfolder "input_images" containing a few .tif images.

You should utilize the same folder structure for your own images; a base folder named whatever you want, and a subfolder specifically named "input_images". You should put your images within "input_images".

In general, you should keep different sets of data within different working directories. For example, maybe you want to count lipid droplets in CARS images, and count stars in a telescope image. You should put the lipid droplet images in "lipid_droplets/input_images/*.tif" and the star images in "stars/input_images/*.tif". Do not mix all the images into one folder. BlobSegmenter attempts to learn what a lipid droplet or a star looks like, and trying to teach it two very different things at the same time might confuse it.

Your images should be 16-bit and have a single channel. If they are not, please convert them appropriately.

Preprocess images

You've installed BlobSegmenter, and you've downloaded some sample datasets. We can finally start counting lipid droplets.

First, open ImageJ. Then, click Plugins -> blobSegmenter -> Preprocess.

A menu should pop up. Click "Set workspace", and select the example working directory that you downloaded.

Click preprocess, and wait until BlobSegmenter has processed all images.

Congratulations, you are done with step one!

Annotate images

Now let's annotate some lipid droplets so that we can teach BlobSegmenter what a droplet should and shouldn't look like

Click Plugins -> blobSegmenter -> Annotate. A new menu should pop up.

Click "Set workspace", and select the example working directory that you downloaded.

Click "Open random image", and a random input image should open up.

Left click on a couple of lipid droplets. A green mask boundary should segment each clicked lipid droplet.

Shift left click a couple areas of the image that have no lipid droplet. A red mask boundary should segment each non-lipid droplet.

Click "Train SVM and apply to image". BlobSegmenter will try to learn what a lipid droplet should look like, and display detections in yellow.

You can now correct mis-classification and re-run the SVM. Alternatively, you can click "Open random image" and annotate some more droplets.

Apply SVM

You've annotated a bunch of positive and negative examples of lipid droplets, and now you want to train and apply an SVM to all input images.

Click Plugins -> blobSegmenter -> Classify. A new menu should pop up.

Click "Set workspace", and select the example working directory that you downloaded.

Click "Train SVM and apply to all images". Wait a bit, until BlobSegmenter tells you it's finished. Now look in the "final_segmentations" subfolder of your working directory. It contains labeled segmentation masks of your lipid droplets.
